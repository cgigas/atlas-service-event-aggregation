ATLAS Event Aggregation Service

Introduction

The ATLAS Event Aggregation Service (UAS) provides a GraphQL API for accessing ATLAS resources and services from the Event Manager Application. The api is available at http://hostname:port/event-aggregation/graphql where port is typically 8080 but may be mapped to 80 in K8s.

Graphql Playground

You can access the EAS graphql API through an embedded Graphql Playground client by accessing the /playground endpoint on this server. Graphql Playground is an excellent tool for testing queries and other features of the hosted schema.
